# MusicDNA - AGENT INSTRUCTIONS

## Mission

MusicDNA is an AI research platform whose goal is to discover the universal principles behind music, emotion and human perception.

The project does NOT exist to generate random prompts.
The project exists to discover measurable musical laws.

## Development Rules

- Never guess.
- Every conclusion must be supported by measurable evidence.
- Every new feature must include a validation method.
- If confidence is low, explicitly report uncertainty.
- Do not remove existing functionality unless replacing it with a tested improvement.
- Prefer small, testable commits.

## Scope

Focus only on the MusicDNA repository.

Do not modify external projects.

## Priorities

1. Measurement accuracy.
2. Reproducibility.
3. Explainability.
4. Research quality.
5. Performance.

## Current Objective

Stabilize all analysis engines before adding new AI features.
